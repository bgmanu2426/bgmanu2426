# My Portfolio Website

A modern, responsive portfolio website built with React, TypeScript, Vite, and TailwindCSS, featuring animated UI components and dark mode support.

## Features

- 🌓 Dark/Light mode toggle
- 🎨 Responsive design
- ✨ Framer Motion animations
- 🧩 Modular component architecture
- 📱 Mobile-friendly navigation
- 🔍 SEO optimized

## Tech Stack

- ⚛️ React 18
- 🔷 TypeScript
- ⚡ Vite
- 🎨 TailwindCSS
- 📦 PNPM package manager
- 🔄 Framer Motion animations
- 🌐 Responsive design

## Development Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/my-portfolio.git
   cd my-portfolio
   ```

2. Install dependencies:

   ```bash
   pnpm install
   ```

3. Start the development server:

   ```bash
   pnpm dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Code Quality Tools

This project uses several tools to maintain code quality:

### Prettier

Prettier is configured to automatically format code for consistency. Key features:

- Auto-formatting on save (with VS Code extension)
- Consistent code style across the project
- Pre-commit hooks to enforce formatting

You can manually format code with:

```bash
# Format all files
pnpm format

# Check formatting without changing files
pnpm format:check
```

### ESLint

ESLint checks code for potential issues and enforces best practices:

```bash
# Run lint checks
pnpm lint

# Fix automatically fixable issues
pnpm lint:fix
```

### Husky & lint-staged

Pre-commit hooks automatically ensure code quality before commits:

- Formats changed files with Prettier
- Runs ESLint to catch issues
- Prevents committing code that doesn't meet quality standards
